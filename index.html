Diabetic Retinopathy Detection

A Flask-based application called Diabetic Retinopathy Detection uses a pre-trained MobileNetV3 model to categorize high-resolution retinal fundus images into diabetic retinopathy stages. To increase accuracy across a variety of clinical datasets, it incorporates contrast enhancement, resizing, and data augmentation. Developed as a component of my bachelor's thesis, this project showcases extensive expertise in TensorFlow model fine-tuning and medical image analysis, culminating in an interactive web interface with performance metrics, real-time predictions, an intuitive design, and easy-to-use upload functionality.

Roadmap App

Roadmap App is a full‑stack Laravel application paired with Vue 3 and Inertia.js that recreates the Microsoft 365 Roadmaps experience, allowing users to browse, upvote, and discuss feature roadmaps. ExChangePlates is a community-driven project developed using Java NetBeans, aimed at bridging the gap between surplus food and those in need. Through a user-friendly interface, donors can easily register and list available food items for donation, while NGOs can efficiently coordinate pickups and deliveries. This project embodies the spirit of generosity and collaboration, harnessing technology to make a meaningful impact on hunger relief efforts.
